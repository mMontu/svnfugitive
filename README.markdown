# svnfugitive.vim


This is an attempt to make some [fugitive][fugitive.vim] plugin commands work
with SVN/subversion.

[fugitive.vim]: https://github.com/tpope/vim-fugitive

Currently supported:

* <None>

Coming soon:

* `:Git` (without parameters)
* staging map -
* staging map X
* `:Git difftool`
